Веб-приложение: Добро начинается с тебя!
Авторы: Дейкина Софья, Цитцер Никита

Описание идеи:
В веб-приложении люди, которые нуждаются в помощи, могут оставить свои обращения. Эти обращения видят неравнодушные люди,
которые готовы помочь бескорыстно. Тем самым сайт, является связывающим звеном между волонтёрами и людьми, которым нужна помощь.

Цель и задачи:

Цель – создать сайт для адресации обращений от людей к волонтёрам.

Задачи:
- Придумать функционал для заказчика
- Придумать функционал для волонтёра
- Создать шаблон сайта
- Освоить необходимые технологии и методы для написания сайта
- Создать базу данных пользователей и обращений
- Реализовать идеи

Описание реализации:
Программа реализована с использованием классов, модулей, функций и т.д.
Основные классы отвечают за:
- users пользователи
- jobs работа
Папка templates с шаблонами для сайта.
Папка forms с формами для авторизаии, регистрации и тд.
Папка static для хранения css и img фалов.
main.py для реализации основной работы сайта.


База данных с несколькими таблицами:
- jobs хранение информации о работе
- users хранение информации о пользователях


Страницы сайта:
- Главная
- Вход
- Регистрация
- Мои обращения
- Добавление обращения
- Редактирование обращения
- Подробное описание обращения

Было применено несколько библиотек, все применённые библиотеки находятся в requirements.txt.
Основные это:
- flask
- flask_restful
- sqlalchemy
- requests
- flask_wtf

Вывод: в ходе работы был создан сайт для адресации обращений от людей к волонтёрам.
